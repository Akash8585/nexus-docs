# API Keys

Base URL: `http://localhost:8000/api/v1/keys`

API keys authenticate agents and SDK clients. Keys use the prefix `nxs_live_sk_` and are sent as Bearer tokens. Only admins can manage keys.

<Warning>
The plain API key is shown **only once** at creation. Store it securely — it cannot be retrieved later. Revoked keys immediately return `401` on all endpoints.
</Warning>

## GET `/keys`

List all API keys. Plain keys are never returned — only metadata.

**Auth required:** Admin JWT

### Response

`200 OK`

```json
[
  {
    "id": "key_a1b2c3d4e5f67890",
    "name": "demo-pipeline",
    "prefix": "nxs_live_sk_AbCdEfGhIj",
    "created_at": "2026-05-30T12:00:00.000Z",
    "is_active": true
  },
  {
    "id": "key_9876543210fedcba",
    "name": "old-key",
    "prefix": "nxs_live_sk_XyZ1234567",
    "created_at": "2026-05-29T08:00:00.000Z",
    "is_active": false
  }
]
```

### cURL

```bash
curl http://localhost:8000/api/v1/keys \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

### Error codes

| Status | Detail | Cause |
| --- | --- | --- |
| `401` | Missing or invalid JWT | Not authenticated |
| `403` | `Admin role required` | Non-admin user |

## POST `/keys`

Generate a new API key.

**Auth required:** Admin JWT

### Request

```json
{
  "name": "demo-pipeline"
}
```

### Response

`200 OK`

```json
{
  "id": "key_a1b2c3d4e5f67890",
  "key_id": "key_a1b2c3d4e5f67890",
  "name": "demo-pipeline",
  "key": "nxs_live_sk_AbCdEfGhIjKlMnOpQrStUvWxYz1234",
  "warning": "Store this key securely. You will not be able to see it again."
}
```

<Note>
Use the `key` field value as your Bearer token: `Authorization: Bearer nxs_live_sk_...`
</Note>

### cURL

```bash
curl -X POST http://localhost:8000/api/v1/keys \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." \
  -H "Content-Type: application/json" \
  -d '{"name":"demo-pipeline"}'
```

### Error codes

| Status | Detail | Cause |
| --- | --- | --- |
| `401` | Missing or invalid JWT | Not authenticated |
| `403` | `Admin role required` | Non-admin user |
| `422` | Validation error | Missing `name` field |

---

## DELETE `/keys/{key_id}`

Revoke an API key. Revoked keys cannot authenticate.

**Auth required:** Admin JWT

### Request

No body. Pass the key ID (e.g. `key_a1b2c3d4e5f67890`) as a path parameter.

### Response

`200 OK`

```json
{
  "message": "API key revoked"
}
```

The key record remains in the list with `is_active: false`.

### cURL

```bash
curl -X DELETE http://localhost:8000/api/v1/keys/key_a1b2c3d4e5f67890 \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

### Error codes

| Status | Detail | Cause |
| --- | --- | --- |
| `401` | Missing or invalid JWT | Not authenticated |
| `403` | `Admin role required` | Non-admin user |
| `404` | `API key not found` | Invalid key ID |
